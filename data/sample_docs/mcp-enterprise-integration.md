# MCP 기반 사내 시스템 연동

MCP는 Agent와 외부 도구를 연결하는 표준 인터페이스로 사용할 수 있다.

기업 환경에서는 단순 연결보다 다음 요소가 더 중요하다.

- Agent가 어떤 사용자 권한으로 도구를 실행하는지 확인해야 한다.
- Tool schema는 입력, 출력, 권한 범위, 위험도를 명확히 표현해야 한다.
- 내부 ERP, CRM, HR, 제조 시스템 응답은 Agent context에 넣기 전에 표준 형식으로 정규화해야 한다.
- Tool call 결과는 감사로그에 남겨야 하며, 실패 시 재시도 가능 여부를 구분해야 한다.

현대오토에버형 Agent 포트폴리오에서는 MCP 서버, A2A 협업, 지식그래프를 함께 보여주는 것이 효과적이다.
